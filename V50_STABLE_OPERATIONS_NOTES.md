# v50 Stable Operations Update

v50은 v46~v50 로드맵을 묶은 안정판입니다.

## 포함 내용

- v46 자동 운영센터: GitHub Actions, 로컬 동기화, API 키, 주요 리포트 상태 확인
- v47 뉴스 카드: GNews/News API 결과를 초보자용 카드로 정리
- v48 재무·KPI 카드: PER/PBR/ROE/성장률 등 값 존재 여부와 해석 카드
- v49 보유·매도 권장수량: 보유수량/평단/현재가 기반 축소·익절·손절 검토 수량
- v50 안정판 안내: 일반모드/관리자모드 역할 정리와 하루 사용 순서

## 실행

```powershell
.\START_HERE_V50_FIXED.bat
```

동기화 없이 실행:

```powershell
.\START_APP_NO_SYNC_V50_FIXED.bat
```

v50 리포트만 갱신:

```powershell
.\run_v50_daily_update.bat
```
