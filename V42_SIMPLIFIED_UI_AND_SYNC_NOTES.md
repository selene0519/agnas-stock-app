# v42 simplified beginner UI + sync notes

## 핵심 변경
- 매수 판단 화면에서 차트를 제거했습니다. 차트는 `차트·수급 > 선택 종목 차트·수급` 한 곳에서만 봅니다.
- `선택 종목 차트·수급` 화면은 차트 1개만 표시합니다.
- 고점/저점은 확정된 최근 스윙 고점 1개와 저점 1개만 표시합니다.
- 보유·매도 통합 화면에서 포트폴리오 리스크 요약을 제거했습니다.
- 매수금지/추격위험/주의는 일반모드에서 `매수 위험·제외`로 통합했습니다.
- 퀀트 메뉴에 `퀀트 안내`를 추가했습니다.
- GitHub 자동누적 결과를 PC 앱에 가져오기 위한 `sync_latest_from_github.bat`, `start_app_with_sync.bat`을 추가했습니다.

## 사용 추천
- 일반 사용자는 `start_app_with_sync.bat`로 실행하세요.
- 이 파일은 먼저 GitHub 최신 데이터를 pull한 뒤 Streamlit 앱을 실행합니다.
