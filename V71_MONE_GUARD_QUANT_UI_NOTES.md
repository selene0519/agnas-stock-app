# MONE v71 업데이트 노트

## 핵심 수정

1. 오늘 실행 첫 화면 5개 카드 복구
   - 오늘 우선 확인
   - 눌림목 진입 후보
   - 수급 급증 후보
   - 실적·저평가 후보
   - 매수금지·주의

2. Donhyun Stock Guard 장점 반영
   - 시장 가드 카드 추가
   - 시장 국면 / 시장 폭 / 섹터 강도 / 리스크 룰
   - 후보·수급·위험 리포트 기반 섹터 강도 요약
   - 장전/장마감에도 직전 리포트 기반으로 빈 화면 감소

3. QuantAI Pro 장점 반영
   - 카드형 요약 UI 강화
   - 보유·매도 권장수량 카드형 + 원본표 병행
   - 상단 KPI식 요약/가드 카드 스타일 적용
   - 초보자용 다음 행동 문구 강조

4. 뉴스·재무·시장 보강
   - 국장 뉴스는 한국어·국내 증시 키워드 필터를 UI에서 한 번 더 적용
   - 미장 뉴스는 기존 간략 한글 해석 유지
   - 기업분석 카드는 v70 DART/Finnhub/SEC 화면 유지
   - 시장·거시 카드는 v71 시장 가드 + v70 원본 지표를 함께 표시

## 실행 순서

```powershell
.\run_v71_daily_update.bat
.\START_APP_NO_SYNC_V71_FIXED.bat
```

GitHub 동기화까지 포함하려면:

```powershell
.\START_HERE_V71_FIXED.bat
```
