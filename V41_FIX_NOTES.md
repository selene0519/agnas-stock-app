# v41 수정 요약

## 고친 문제

- 일반모드와 관리자모드 메뉴 중복 재정리
- 매수금지/추격매수/리스크 우선 후보를 일반모드에서는 `매수 위험·제외` 하나로 통합
- 관심종목 차트와 선택 종목 수급 차트 중복 제거: 일반모드는 `선택 종목 차트·수급` 중심
- 차트 고점/저점 마커 기본 OFF, 켜도 최근 고점 1개·저점 1개만 표시
- 예산·권장수량 화면에서 계좌/현금/리스크 기준을 직접 저장 가능
- 커스텀 스크리너가 빈 후보 파일만 보고 0건이 되는 문제 완화
- 선택 종목 수급표를 초보자용 해석/다음 행동 중심으로 변경
- 자동누적기에 v37 운용표와 v40 분석표 갱신을 포함

## 여전히 사용자가 해야 하는 일

- 뉴스/LLM/DART 등 외부 API 키 발급 및 `.env` 입력
- 보유종목 수량/평단가 입력
- 노트북이 꺼져도 자동누적하려면 클라우드 배포 필요
