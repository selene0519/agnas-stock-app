# 뉴스 API / LLM 연결 안내

이 앱은 코드상으로 뉴스 캐시와 요약 화면을 지원하지만, 실제 외부 뉴스 본문 요약은 API 키가 있어야 자동화됩니다.

## 사용자가 해야 하는 일

1. 뉴스 제공 API 키 발급
   - Finnhub, NewsAPI, GNews, SerpAPI 등 중 하나를 사용할 수 있습니다.
   - 무료 플랜은 호출 횟수와 본문 제공 범위가 제한될 수 있습니다.

2. LLM 요약 API 키 발급
   - Claude, OpenAI, Gemini 등 중 하나를 사용할 수 있습니다.
   - 앱은 API 키가 없으면 제목/description/로컬 캐시 기반 요약만 표시합니다.

3. 앱 폴더에 `.env` 파일 생성
   - `.env.example`을 복사해서 `.env`로 이름을 바꿉니다.
   - 발급받은 키를 넣습니다.

## 앱이 자동으로 할 수 있는 일

- 키가 설정되어 있으면 자동누적기가 뉴스 캐시를 갱신할 수 있습니다.
- 중복 뉴스 제거 후 종목별 요약으로 표시할 수 있습니다.
- API 키가 없으면 가짜 뉴스를 만들지 않고 “뉴스 수집 안 됨”으로 표시합니다.

## 주의

뉴스 기사 본문은 매체 정책, paywall, robots 정책에 따라 수집이 제한될 수 있습니다.
본문 전체를 저장하거나 장문 복사하는 방식은 권장하지 않습니다.
