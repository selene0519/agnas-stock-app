# V66 MONE UX Cleanup

## 반영 내용
- 앱 표시 이름을 MONE으로 변경했습니다.
- 일반모드 `오늘 실행`은 `오늘 우선 확인` 한 화면으로 단순화했습니다.
- `매수 후보 5분류`는 중복 위험 탭을 제거해 `매수 후보 4분류`로 정리했습니다. 위험 후보는 `매수 → 매수 위험·제외` 한 곳에서만 봅니다.
- 뉴스 카드는 국장/미장 필터별로 따로 표시하고, AI API 없이도 규칙 기반 한글 해석을 제공합니다.
- 기업분석/시장거시/종목내러티브는 실제 데이터가 없으면 `운영 준비/데이터 필요`로 명확히 표시합니다.
- 퀀트 안내·복기/백테스트/몬테카를로/옵션은 관리자모드의 `퀀트·복기`로 이동했습니다.
